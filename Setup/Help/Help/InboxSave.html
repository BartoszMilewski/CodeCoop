Saving File from within the Inbox

When you are not in auto-sync mode, incoming scripts will be listed in the Inbox tab. When you select an incoming script, the files affected by it will be listed below in the details pane. Select some or all files in the details pane and click the save button in the toolbar. You have the option to save the file(s) in the state they were before this incoming change, or with the incoming changes applied.
