Simple Mapi and Mapi Compliant Email programs

Code Co-op integrates with the following Email clients:

  • Outlook
  • Outlook Express
  • Eudora
  • Lotus Notes
  • Netscape Navigator

Mozilla Thunderbird doesn't work with Code Co-op because Mozilla has only done a partial Mapi implementation.


Consider SMTP/POP3 Direct Connection

If your email program is not supported, consider setting up Code Co-op with a direct SMTP/POP3 connection. Having a dedicated email address for Code Co-op is ideal. Consider Gmail.
