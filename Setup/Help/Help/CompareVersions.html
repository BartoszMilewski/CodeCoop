Comparing Versions

You can visually compare any two versions of the project.

Comparing Historical Version with Current Version

Go to History tab and select the version you're interested in comparing. By just highlighting that version (script), you will see, in the lower pane, the list of files changed by that script. After selecting Show Changes Since this Script from the Selection or the Context menu, you will see the list of all the files that have been changed by the selected script and all script that follow it, up to the current version. Double-click a file to view the cumulative changes in a differ.

Notice that scripts marked with boxed icon (the range) would be undone if you were to Branch, Restore, or Export from the selected version.

Comparing Two Arbitrary Versions

If you want to compare two arbitrary versions, select one of them and then Ctrl- select the other--the range. The Details pane will list all the files that were changed by the scripts marked with the boxed icon. Double-click a file to view the changes in a differ.

Note: There is a subtle difference between selecting the last script in the history and the Current Version, as the endpoint for comparison. When the Current Version is selected, the comparison will also take into account the changes in the currently checked out files.
