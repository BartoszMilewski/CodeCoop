Licensing

After the trial period, if you want to continue using Code Co-op, you will have to purchase a license from Reliable Software. You enter the license information in the dialog displayed through License in the Program menu. You will need at least as many licenses as there are Voting Members in a given project (Observers don't need licenses). The same license can be used in any number of projects on a given workstation. If there aren't enough licensed seats in a project, Code Co-op will force the status change of some members to Observer. These members can recover their voting rights after purchasing and entering their licenses.


Distribution License

In situations where you are using Code Co-op to distribute source code to clients through Distribution Projects, you will need a distribution license for each recipient (client). These licenses are sold at a discount. More information can be found on our web site www.relisoft.com. (Our RSWL library is distributed that way.)
