Editing an Incoming Script

When you have an incoming script in your Inbox, you have the option to make edits to the affected files as you review the changes.


To edit the sync script:

  • Select the script in the Inbox
  • From the Selection or context menu select Edit Sync
  • You will be taken to the Synch Merge tab where you can double-click each file and make edits using the visual differ
  • Click the Accept Changes button in the toolbar or call it from the context or Selection menu to sync the incoming sync changes
  • Go to the Check-in tab to Check-in and propogate the changes just made to other members of the project
Note: If you are using the built-in differ, you can also edit sync changes from the History Area, after they've been accepted. You won't be editing the script but the Current version of the file shown side-by-side with the script's changes.
