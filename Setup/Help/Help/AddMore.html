Bulk Addition of Files

Files can be added to project during its creation or at any later time. You can add files individually or in bulk. When you select Add More Files from the Project menu, Code Co-op will rescan the whole project and list all non-project files organized by file extensions. You can select whole groups or individual files and add them to project. Of course, you'll have to check in the newly added files in order for other project members to see them.

The same operation, Add More Files, only on a smaller scale is available from the Folder menu. Code Co-op will scan only the selected folder and its sub-folders.
