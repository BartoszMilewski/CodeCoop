Executing a Script

The Inbox area lists all the incoming scripts. The Inbox is intelligent enough to recognize which script should be executed next. This next script in line is marked using this icon. Selecting any script will list the affected files in the details pane below. Sometimes you'll see a script marked "missing". If you have a missing script, Code Co-op will automatically send re-send requests to other project members.


Execute scripts by doing one of the following:

  • Click the Execute next script button (it is only active when there is something to execute)
  • Select Sync Next Script entry in the Selection menu
  • Use the keyboard shortcut to execute all incoming scripts: Ctrl-S

By executing a script you are applying changes from the script to the files in your project. Sometimes Code Co-op detects a conflict between script changes and your local edits--you will be notified whether you need to attend to a merge. Sometimes there is a conflict betwee scripts that miss each other. In that case you will be notified that a branch was created and you should merge it back.
