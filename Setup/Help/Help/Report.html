Creating Reports

Code Co-op has rudimentary reporting capabilities. If you select Report from the Selection or All menu, you'll be able to save the data from the current view to a file. This is particularly useful in History view. The file contains tab-separated columns, best viewed by importing it into a spreadsheet.

For diagnostics purposes, Code Co-op has the Save Diagnostics... item in the Help menu. Reliable Software support might ask you to email the results of diagnostics in order to help you resolve a problem.
