Importing Project History

When you join a project, your project history is initially empty. If you need access to the history of the project prior to your joining, you can import the history from another project member.

First, have another project member export their history and send it to you. Generally, the Admin has the longest project history. Then, when you receive the project history, incorporate it into your project by selecting Import History from the History menu (you will be prompted for the file name).


Note: Your history and the imported history must partially overlap for the Import to work. In particular, you cannot import a history file created before you joined the project.
