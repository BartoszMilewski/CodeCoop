Project Options

Using the Project > Options menu item you can customize how each project handles incoming scripts and checked-out files. These options can also be selected while you are setting up a new project, regardless of whether you are creating or joining it.

General Options

The first option lets you set the Auto-Sync feature to automatically sync incoming scripts. If you are the project Administrator, you can also set a project to auto-accept Join requests. If you want to keep your files checked out after each check-in, mark the last option.

Distributor options

The Distributor tab is for special distribution project. Go to our Distribution Project page for more information.
