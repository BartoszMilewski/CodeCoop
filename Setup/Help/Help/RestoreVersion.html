Restoring old Versions of Files

There are two ways to recreate a particular version of a file (or files).

From History Tab

  • Select the version in the history
  • Select the file in the lower pane
  • Select Restore file version from the Selection menu.
    • The file will be restored to the state before the execution of the selected script.

    Instead of restoring in-place, you may also save the old version of the file to a different locaction using the Save file version menu item.

    From the Files Tab

    • Select the file or files or interest. To select multiple files, hold Ctrl and select the files
    • Right-click and select Show History from its context menu or click , the Show History button
    You'll be switched to the History tab with the appropriate file filter selected.

    Alternatively, you can set up the History File Filter directly in the History tab:

    • In the combo box at the top, type in the name of the file. (see Looking at file versions)
    • All the scripts that carried changes to this file will be listed.
    • Select the version (script) you're interested in.
    • Double-click the file in the Details pane to view the changes.
    • To save this version of the file, click the save button in the Details tool bar
