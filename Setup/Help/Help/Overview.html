Code Co-op Documentation Overview

Code Co-op is a version control system, whose unique characteristics are

  • It doesn't require a central server (code repository is replicated on each machine)
  • It supports collaboration using e-mail (SMTP/POP3 or MAPI), LAN, and VPN
  • It supports distributed collaboration
