Open Invitation

In Code Co-op Pro, it is possible to create a project invitation without sending it through the usual channels. Such an invitation may be stored on a disk, LAN, or on an FTP site.

The invitee can open or download such an invitation using the Project>Open Invitation menu item. The following dialog appears:

Use the Browse buttons to locate the file, either locally, on the LAN, or on an FTP server.

Important: The invitation can only be used once, by the intended recipient. The attempt to accept the same invitation twice, even after defecting, will result in an unusable membership database.
