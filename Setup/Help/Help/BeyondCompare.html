Beyond Compare

The distributon of Code Co-op includes the Beyond Compare (BC) Differ/Merger, which is a light version of Scooter Software's Beyond Compare product. This version is limited to comparing and merging text files, as opposed to binary files and whole directories.

The Differ

For showing file differences, you have a choice of using the BC Differ or a built-in differ. Each has its own advantages and disadvantages. The built-in differ, for instance, is useful when besides viewing differences between historical version (either in Inbox or History view), you would also like to edit the current version of a file.

The Merger

The BC merger distributed with Code Co-op can be used for inter-project merges or local branch merges.
