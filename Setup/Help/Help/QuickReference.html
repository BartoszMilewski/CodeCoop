UI Quick Reference

Main Window

(Click a tab in the picture to jump to its description.)

The main window of Code Co-op contains tabs corresponding to different working areas:

  • Files
  • Check-in
  • Inbox
  • History
  • Projects

Dynamically Created Tabs

  • Synch Merge - For incoming synch merges
  • Branch Merge - For local branch and inter-project branch merges
  • Browser - For viewing local wiki sites

Directly below the tabs is the toolbar. Its contents are context-sensitive. Different buttons are visible in different areas. Some buttons are enabled only when there is a selection in the main window. All commands in the toolbar are also available from the menus.

The status bar at the bottom displays the status of Code Co-op, short help when you select a menu item, and a progress bar during long operations.


Details Pane

A Details pane appears in the Files, Inbox, History, and Projects tab.
