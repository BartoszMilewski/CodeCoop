Adding Existing Files and Folders

If a file has been created outside of Code Co-op's control and saved within a project tree, it doesn't automatically become part of your project. You have to Add it to the project. From within the Files area, select the file(s) and click the Add To Project button. Other members of the team will learn about this addition after you make your next check-in.

You may also add a whole folder to the project. When you select a folder and click on the Add To Project button, you will be given a choice as to which files in that folder (and its sub-folders) to add. You'll be able to select groups of files to be added either by extension or individually.
