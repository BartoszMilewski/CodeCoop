Move Projects from Current Computer to New Computer

(This feature is not available in Code Co-op Lite)

If you want to move the whole Code Co-op database (all projects and their histories) from your current computer to a new computer follow these steps:

  1. On the source computer
    • Create a full Code Co-op backup using Tools>Create Backup. Move the resulting file to the new computer.
    • If the Disptatcher is using e-mail, make sure you know how to configure it on the new machine.
  2. On the target computer
    • You should have a fresh installation of Code Co-op with no projects present
    • Select Tools>Restore From Backup and select the backup file created in step 1
    • The process of restoring projects may take a very long time with very slow progress. Be patient!
    • For each project a verification request is sent to another project member--please select them from the list
    • After the procedure completes, check-ins will be blocked in those projects that haven't received back verification scripts. (If necessary, you can unblock them using Project>Request Verification and clicking on the checkbox "Don't block check-ins".)
  3. On the source computer (after the process finished successfully on the target computer)
    • Uninstall Code Co-op. Make sure to select the option of not defecting from projects!

See also: Create Backup and Restore from Backup for explanation of Backup and Restore.
