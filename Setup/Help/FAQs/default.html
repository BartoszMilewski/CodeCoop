Setting your email program as the default email client


If you have multiple email clients on your computer, you need to make sure that the email client that you want to use with Code Co-op is set as the default. There are a few ways how this may be done.


First try the easiest way: launch your email client. Most of them (e.g. Outlook, Outlook Express, Eudora, Netscape) perform a check at startup to see whether they are set as default. If not, they propose to make them default.

If nothing like this happens you need to find an option in your email program that makes it the default. For instance, in Outlook Express, go to Tools.Options dialog and select Make Default at the bottom of the General page.


Alternatively, Internet Explorer users can launch IE and select Tools > Internet Options. Select the Programs Tab and verify that the correct email client is selected.


Regardless of the email program there is always an option to change the default email client through Control Panel. Select Internet Options > Programs tab and select your favourite email client from the list.
