Working with Visual Studio.NET and Visual C++ 6.0


Code Co-op integrates closely with Visual Studio.NET and Visual C++ 6.0, making it easy to add version control to your development projects. Once you've connected your development projects to Code Co-op projects, you'll be able to check out files automatically as you work on them.
