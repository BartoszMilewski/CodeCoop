Can I use AOL, Yahoo, Gmail, Hotmail, or other free email accounts with Code Co-op?


You can use almost any type of email account with Code Co-op. As of version 5.0, you can now connection directly to your email account through POP3/SMTP. Most email accounts support this protocol, but only some provide this free of charge.

Here is a list of the email accounts we're aware of that support this protocol.

  • Gmail, Hotmail and Yahoo have free SMTP/POP support
    • Gmail - Gmail support is built-in. See our tutorial for set-up instructions.
    • Hotmail (Windows Live)- Configure SMTP/POP3 to use Hotmail.
    • Yahoo - In order to gain access to Yahoo through SMTP/POP, you will need to install YPOPS. YPOPS is an open source project application. See our page on Setting Up Yahoo
  • AOL SMTP/POP support requires purchase
    • AOL - There are two products which will give you SMTP/POP support with AOL: Email2Pop and eNetBot. The cost for either of these is quite modest at under $20.
