Joining a project: New member

Code Co-op Lite

This dialog shows Chad joining a project where the Admin is an Email peer.

  • First, for routing purposes, select the appropriate option:
    • When the project Admin is on your computer, select the radio button "To me (I want to have more enlistments in this project)".
    • When the project Admin is a different email peer, select the radio button "To an e-mail peer using the following address (type in or select)" and either select an email address from the drop down or type in a new one.
  • Next, type the exact name of the project.
  • Then, designate where to put the copy of the project on your machine. Select a convenient directory. After the join process is completed, you will have an exact copy of the project in that directory.
  • Finally, fill in your personal data and click OK.

The Join request is sent out automatically by the dispatcher.
