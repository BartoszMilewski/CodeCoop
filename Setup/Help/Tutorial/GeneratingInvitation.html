Generating an Invitation: Administrator


The project Administrator (usually the person who created the project in the first place) can invite other Code Co-op users to the collaboration.

If you are the administrator, you invite a person to the project by selecting Project > Invite menu item. The invite dialog has more options in the version Pro than in the version Lite of Code Co-op. Select the next page depenging on what version you are using.
