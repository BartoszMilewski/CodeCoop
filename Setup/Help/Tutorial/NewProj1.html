Creating a new project: Dialog

In the New Project dialog:

  • Specify a local project path. If you already have a body of source code in a directory on your machine, enter the location of that directory. Its subdirectories will be automatically included in the project. If you're creating a project from scratch, type in the name of the project's top-level directory. If it does not exist, Code Co-op will create it for you.
  • Specify the project name.
  • Input your personal data: your name, and an optional comment (for instance, your phone number)
