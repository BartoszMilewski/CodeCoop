Joining a Project


As an alternative to being invited by the administrator, you can request to Join a project. A Join is initiated by the new member — the person wanting to join the project — instead of the project Admin.


To join an existing project, select Project > Join from the menu. The Join dialog has more options in the version Pro than in the version Lite of Code Co-op. Select the next page depenging on what version you are using.
