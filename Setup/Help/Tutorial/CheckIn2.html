Reviewing File Changes

Before you check in your changes, it's a good idea to review them first. A lot of bugs are prevented that way. Double-click on any edited file in the Check-In Area, and Code Co-op will open a Differ, in a two-pane window. (Here you can see the built-in differ, but you can switch to the Beyond Compare differ, which comes bundled with Code Co-op, using the Tools>Differ menu item.)

The left pane has two tabs:

  • Current tab shows the current file with your changes applied. Additional changes can be made in the Current tab, which serves as an editor. (Click Refresh in the toolbar to update the right pane after such edits.)
  • Before tab shows the file as it was when you checked it out--before your changes.

The right pane shows all the changed lines.
