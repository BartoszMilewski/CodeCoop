Starting Collaboration: Overview

Project Invitation

Here is the review of the invitation process:

Who Action Result
Admin Create Invitation Full sync is packed and sent to new member
New Member Receive Invitation New member receives invitation and designates project location. Project files are recreated.

Project Join

Here is the review of the Join process:

Who Action Result
New Member Join Project Join Request script is sent to the Admin
Admin Execute Join Request script Full Sync script is sent to New Member
New Member Execute Full Sync script Project files recreated
